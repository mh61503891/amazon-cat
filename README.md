# Amazon Cat: A command line tool to search items in Amazon

[![Gem Version](https://badge.fury.io/rb/amazon-cat.svg)](https://badge.fury.io/rb/amazon-cat)

## Usage

```bash
gem install amazon-cat
```

```bash
amazon-cat help
```
